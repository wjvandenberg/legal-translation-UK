## Notes

- **Código da Propriedade Industrial (CPI)**: Portugal's Industrial Property Code — covers
  patents, trade marks, designs, and trade secrets. Reference as "Industrial Property Code
  (CPI)" on first mention.
- **Código do Direito de Autor e dos Direitos Conexos (CDADC)**: Portugal's Copyright
  and Related Rights Code. Reference as "Copyright Code (CDADC)" on first mention.
- **INPI (Instituto Nacional da Propriedade Industrial)**: Portugal's national intellectual
  property office. Reference as "Portuguese National Institute of Industrial Property
  (INPI)" on first mention.
- **"Contrafação"**: The Portuguese term for IP infringement, particularly of trade marks
  and patents. Translate as "infringement" or "counterfeiting" depending on context.
- **"Direitos de autor" — no assignment in Portuguese law**: Like German Urheberrecht,
  Portuguese copyright (direito de autor) cannot be fully assigned — only exploitation
  rights (direitos patrimoniais) can be transferred. Moral rights are non-transferable. Flag
  this distinction in translation.
- **"Marca" spelling**: Use "trade mark" (two words, UK) or "trademark" (one word, US)
  depending on target audience. Default to UK (two words).

## Notes

- **CNPD (Comissão Nacional de Proteção de Dados)**: Portugal's data protection authority.
  Reference as "Portuguese Data Protection Authority (CNPD)" on first mention.
- **RGPD**: The Portuguese abbreviation for the GDPR. Use "GDPR" in English translation
  but note the Portuguese abbreviation on first mention.
- **"Subcontratante" (RGPD)**: In GDPR Portuguese terminology, "subcontratante" means
  data processor (not subcontractor in the usual sense). This is a frequent source of
  confusion — always translate as "processor" in GDPR context.
- **Lei n.º 58/2019**: Portugal's national law implementing the GDPR. Reference as
  "Portuguese Data Protection Act (Lei 58/2019)" on first mention.
- **"Aceitação" in IT contracts**: Portuguese IT contracts structured as "empreitada"
  (works contracts) require formal acceptance (receção) which triggers warranty and
  payment obligations. Translate as "acceptance" or "user acceptance testing".
