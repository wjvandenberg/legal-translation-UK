# Italian — IP, Technology and IT Sub-Lexicon

Combines Italian-language IP/technology and IT/SaaS vocabulary. Part 1 covers copyright, industrial property (patents, designs, trade marks), trade secrets and technology transfer. Part 2 covers software licensing, SaaS/cloud services, implementation projects, SLAs, data processing and cybersecurity.

## Notes

- **"Proprietà industriale" vs "diritto d'autore"**: Italian law divides IP into industrial property (D.Lgs. 30/2005) and copyright (L. 633/1941). English uses umbrella term "intellectual property" covering both.

- **"Contraffazione"**: In patent/trademark context means "infringement". In criminal/goods context (counterfeit products) also "contraffazione". Translate contextually.

- **"Licenza" to "licence/license"**: Italian uses single term "licenza" for all contexts. English requires "licence" (UK noun), "license" (US noun/verb). Default to UK noun.

- **"SIAE"**: Italian copyright collecting society collecting and distributing royalties for authors/publishers. Not equivalent to foreign societies but serves similar function. Keep as "SIAE" with explanation in translations.

- **"Marchio" spelling**: Always "trade mark" (UK) or "trademark" (US) depending on target audience. Default to UK (two words).

- **Italian legislation references**: When translating Italian IP agreements, CPI, SIAE, and other Italian-specific references belong in translation notes, not main contract text, as English-language contracts typically reference international frameworks or target jurisdiction's law.

## Contextual Translation Notes

- **"Crediti di servizio"**: In SaaS context = "service credits" (invoice discount for failures). In traditional outsourcing = "penali" = "liquidated damages" (contractual damages). Not interchangeable.

- **"Licenza"**: Single Italian term; always translates to UK "licence" (noun) or "license" (US). Default UK noun form.

- **"Livello di servizio"**: Contract element typically includes uptime %, exclusions (scheduled maintenance, force majeure), measurement methodology, and remedy (service credits).

- **"Ambiente"**: Common to specify both production (live users) and staging (testing) environments; clarify in translation which applies to warranties, backups, SLA.

- **"Aggiornamento vs Upgrade"**: Italian uses single term but context matters—minor patches = updates; major releases = upgrades. Licensing implications differ.

- **GDPR-specific terms**: Italian uses "trattamento" (processing); English GDPR context requires specific terms—"data controller", "data processor", "personal data"—with definitions in DPA.

- **"Collaudo"**: Technical acceptance testing; often formal sign-off triggering payment milestone in development contracts.

- **"Open source"**: Italian agreements often address via "componente open source"; ensure translation specifies licence type (copyleft vs permissive) and compliance obligations.

- **"Proprietà dei risultati"**: In development context = ownership of developed IP; often allocated to hirer in work-for-hire structure but negotiable.

- **"Escrow"**: Used for source code escrow (holding code released if developer fails); third party typically escrow agent; common in enterprise software.

- **Data transfer terminology**: "Migrazione" = data migration; "esportazione" = export; "portabilità" = portability right; all distinct concepts in contract.

- **SLA metrics**: Italian contracts often specify penalty tiers (e.g., 1% credit 99-99.5%, 2% credit 98-99%); translate into service credit schedule clearly.
